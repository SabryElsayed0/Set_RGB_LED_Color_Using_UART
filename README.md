# Set_RGB_LED_Color_Using_UART
## Overview:
In this project, you can send messages from your PC keyboard by using Docklight simulation tool to communicate in serial with Atmega32 using UART Communication protocol. The messages will be RGB LED COLORS such as (RED, BLUE, GREEN, YELLOW, WHITE, PINK, PURPLE). For example, if you send RED the RGB will be lighted with red color While if you send BLUE it will be lighted BLUE, and so on. If you send anything other than colors error message will be printed on LCD.
### Drivers: 
DIO, RGB, LCD, UART, TIMERS.
#### Microcontroller:
ATmega32
